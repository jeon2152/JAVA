PURPOSE OF PROJECT: Creating an interactive game

HOW TO START THIS PROJECT:

USER INSTRUCTIONS:

- Use keypad Right, Left, Up and Down to move the green car.
- Try to catch gift boxes to earn 1 point for each box.
- AVOID blue vehicles, trees and snakes. When the green car collides with a vehicle or a tree or a snake, the game stops and you lose.
- You are allowed to move outside of while lines. Watch out for the trees and snakes!
- Time is ticking. When you earn 25 points or more til the time is up, you win the game. If you get less than 25 points, try it again!
